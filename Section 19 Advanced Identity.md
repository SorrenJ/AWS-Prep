# Directory Services

AWS doesn't have active directories  but you can extend active directories using AWS directory services

It provides directory solutions such as AWS Managed Microsoft AD, AD Connector, and Simple AD, enabling seamless integration with AWS services, including Amazon EC2, WorkSpaces, and Amazon RDS. Directory Service simplifies user management, authentication, and group policy administration across hybrid IT environments.

# Quiz

Question 1:
A company would like to use their on-premises Microsoft Active Directory to connect to its AWS resources. Which service can it use?

Directory Services

Makes it easy for you to setup and run directories in the AWS cloud, or connect your AWS resources with an existing on-premises Microsoft Active directory

Question 2:
which AWS service allows you to create temporary, limited-privilege credentials for your AWS resources?

Secuirty Token Service (STS)

A web service that enables you to request temporary, limited-privilege credentials for AWS identity and Access Management (IAM) users or for users that you authenticate (federated users)


Question 3:

A company just created a new mobile application and wants to add a simple and secure user sign-up, sign-in, and access control. Which AWS service can it use?

Cognito

let's you add user sign-up, sign-in, and access control to your web and mobile apps quickly and easily

Question 4:

A company would like to centrally manage access to multiple AWS accounts and business applications. Which service can it use?

IAM Identity Center

an AWS service that enables you to makes it easy to centrally manage access to multiple AWS accounts and business application and provide users with single sign-on access to all their assigned accounts and application from one place
